# Fliprop Phase 1 Imagegen Prompts

Use built-in `image_gen` when available. Generate three variants for each prompt, then move the selected final into the destination path listed below.

Sharper fintech direction for this round:
- cleaner contrast, cooler white surfaces, deeper ink framing
- restrained teal as the main action accent
- soft editorial lighting, not glossy marketing lighting
- avoid warm-beige drift, loud gradients, heavy 3D chrome, and fake readable UI text
